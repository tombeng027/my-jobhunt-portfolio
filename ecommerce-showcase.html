E-commerce Showcase

This page demonstrates a front-end e-commerce experience with product filtering, cart management, and checkout summary logic.

← Back to Portfolio

Features Demonstrated

  • Live product and category data fetched from the FakeStore REST API
  • Product search and category filters (categories populated dynamically from API)
  • Dynamic product rendering with images, badges, and stock levels
  • Cart quantity controls with stock-limit validation
  • Live subtotal, shipping, and total calculations
  • Cart state persistence using localStorage
  • API response cached in localStorage — products and categories load from cache if the API is unavailable

Technical Stack Used

HTML5 CSS3 Vanilla JavaScript (ES6+) Vite Responsive Design REST API Integration Fetch API / async-await Local Storage API
